# 📧 Email Spam Detection using Machine Learning

Hi, I’m **Ayesha Ansari** 👋

This project is part of my **Data Science Internship at Oasis Infobyte (OIBSIP)**.
In this task, I built a Machine Learning model to classify emails as **Spam or Not Spam**.

---

## 📌 Project Overview

Email spam detection is an important application of Machine Learning.
The goal of this project is to automatically detect whether a message is spam or not based on its content.

---

## 🚀 What I Did

* Loaded and cleaned the dataset
* Converted text data into numerical format using CountVectorizer
* Split the dataset into training and testing sets
* Trained a Machine Learning model
* Tested the model and checked its accuracy
* Predicted custom messages

---

## 🧠 Technologies Used

* Python 🐍
* Pandas
* Scikit-learn
* CountVectorizer

---

## 🤖 Model Used

I used **Logistic Regression** for classification.
This model helps in predicting whether a message is spam or not based on learned patterns.

---

## 📊 Results

The model achieved good accuracy on the test data and was able to correctly classify spam and non-spam messages.
